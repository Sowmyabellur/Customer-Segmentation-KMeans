# Customer-Segmentation-KMeans



## Objective
To segment customers into distinct groups based on income and spending behavior.

## Dataset
Mall Customers dataset.

## Methodology
- Feature selection
- Data scaling
- Elbow method to find optimal clusters
- K-Means clustering
- Visualization of clusters

## Results
Customers were segmented into five distinct clusters with clear spending patterns.

## Business Use Case
Helps businesses perform targeted marketing and personalized promotions.

## Conclusion
This project demonstrates the application of unsupervised learning for customer behavior analysis.
